# ConUHacks VII (2024)

## Utilisation

### Client

Utiliser Docker ou faire

```bash
npm run dev
```
dans le répertoire `client/volt-client`

### Serveur

Utiliser Docker (soon TM) ou faire

```bash
uvicorn server:app
```
dans le répertoire `server`. Assurez vous d'installer les dépendances avec un venv
